# Twitter / X Post

Posts tweets and threads on X (formerly Twitter) via the web interface.

## Triggers
- "tweet [text]"
- "post on twitter"
- "post on x"
- "twitter post [text]"

## Example Commands
- "Tweet: Just shipped a new feature!"
- "Post on X: Loving the new update from @cyclop_one"
- "Tweet a thread about today's product launch"

## How It Works
1. Opens x.com in Chrome or the default browser
2. Clicks the Post/compose button
3. Types the tweet text
4. Clicks Post to publish

## Notes
- 280 character limit per tweet
- For threads, posts multiple tweets sequentially
- Confirms before posting if content may be sensitive
